# 正EV交易决策台

一个用于**下单前**的支撑阻力、进出场与风险检查工具。它不负责预测涨跌，而是把多个交易框架对“位置、触发、失效和目标”的判断放在同一张地图里。

## 当前能力

- 多周期行情：1m、5m、15m、1h、4h、1d
- 市场：主流加密资产；美股与大宗使用浏览器可访问的数据源
- 下单前位置地图：将多种价位来源聚合为支撑/阻力区
- 条件式情景：支撑确认、阻力确认、突破回踩、跌破反抽
- EV 门控：方向、位置、赔率同时成立才允许记录计划
- 风险仓位：按账户规模与单笔风险计算名义仓位
- 多流派评审团：可选择专家并调整权重
- 本地交易日志：用 R 值记录结果并计算纪律分

## 17 个分析视角

| 类别 | 方法 | 主要回答 |
|---|---|---|
| 趋势结构 | 趋势跟踪、道氏理论、一目均衡 | 当前更适合顺势、逆势还是等待 |
| 价格行为 | Al Brooks 价格行为 | 趋势K、交易区间与触发K如何组合 |
| 量价行为 | 威科夫、成交量分布、锚定 VWAP | 市场接受价、成本区与区间边缘反应 |
| 结构与流动性 | SMC、ICT 流动性 | BOS、扫损、前高前低与 FVG |
| 东方结构 | 缠论 | 中枢、背驰与潜在买卖点 |
| 价位工具 | 支撑阻力聚类、斐波那契 | 哪些价位是高汇合决策区 |
| 统计指标 | 均值回归、动量因子 | 行情是否偏离均值或存在持续动量 |
| 执行与环境 | 网格、资金费率情绪、宏观 | 当前策略是否适配、是否存在拥挤风险 |

不同方法并非完全独立。工具会展示汇合来源，但不会把高度相关的多条线解释成多份独立证据。

## 价位地图

价位候选来自：

- 多周期摆动点聚类
- 唐奇安区间与 60 根摆动高低
- 前周期高、低与经典枢轴
- EMA20 / EMA50
- Bollinger 上、中、下轨
- Fibonacci 0.382 / 0.5 / 0.618 / 0.786
- 成交量 POC / HVN（由 K 线典型价近似）
- 重要波段锚定 VWAP
- 整数心理关口

相近价位按 ATR 容差聚合为“区”，再按照来源权重、重复测试与来源多样性评分。分数表示**汇合强度**，不是胜率。

## 决策顺序

1. 先看多周期行情环境。
2. 再看现价离最近强支撑/阻力多远。
3. 选择主情景与备选情景。
4. 等待收回、拒绝、突破回踩等明确触发。
5. 把止损放在结构失效区外。
6. 目标取下一处有意义的汇合区并计算 R:R。
7. 方向、位置或赔率任一不成立，不下单。

## 数据与限制

- 全部计算在浏览器内完成，偏好与交易日志保存在 localStorage。
- 浏览器端公共 API 可能受地区、CORS 或限流影响。
- K 线成交量分布是近似值，不等同交易所逐笔成交或完整订单簿。
- Brooks、ICT、威科夫和缠论等依赖上下文的框架被规则化简，仅适合作为结构化检查项。
- 仅对已有历史数据的“专家 × 标的 × 周期”显示回测；没有数据时明确标记待接入。
- 历史回测和汇合分数均不保证未来表现。本项目是分析与教育工具，不是投资建议。

## 运行

